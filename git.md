Для игры в викторину по git:

1. Уметь работать локально используя команды:
   1. git log
   2. git checkout
   3. git add
   4. git commit
2. Уметь работать с ветками
   1. git branch
   2. git merge
3. Уметь работать с удаленным репозиторием
   1. git clone
   2. git push
   3. git pull
4. Уметь отправлять pull request
